facebook4j-twitter4j-vraptor
===========

Facebook4J and Twitter4J with Spring MVC. Based on [this](https://github.com/roundrop/facebook4j-oauth-example/) and [this](https://github.com/yusuke/sign-in-with-twitter/) examples.
